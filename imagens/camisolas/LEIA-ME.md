# Fotos das Camisolas

As camisolas são carregadas a partir do Supabase (tabela `camisolas`, campo `imagem_url`).

Podes guardar as imagens aqui localmente e apontar o `imagem_url` na base de dados para:
```
imagens/camisolas/NomeDoFicheiro.jpg
```

## Estrutura sugerida
```
imagens/camisolas/
  Sporting_Home_2024.jpg
  Sporting_Away_2024.jpg
  Porto_Home_2024.jpg
  ...
```
